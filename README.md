# 🥤 Spylt Drink Website Clone

A modern and visually engaging **Spylt drink website clone** built using **Next.js**, **GSAP**, and **Tailwind CSS**. This project focuses on smooth animations, responsive design, and an immersive UI experience.

---

## 🚀 Tech Stack

- ⚡ **Next.js** – React framework for fast and optimized web apps  
- 🎨 **Tailwind CSS** – Utility-first CSS framework for styling  
- 🎬 **GSAP (GreenSock Animation Platform)** – Advanced animations and scroll effects  

---

## ✨ Features

- 🧊 Smooth scroll-based animations using GSAP  
- 📱 Fully responsive design for all screen sizes  
- 🎯 Modern UI inspired by Spylt branding  
- ⚡ Optimized performance with Next.js  
- 🎥 ScrollTrigger-based section animations  
- 🧩 Component-based architecture
